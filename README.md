- Shop für Upgrades/Leben --> Währung sind Coins bzw. Pkt
- mehr Items (Autonom/Drohne (10s), Christiano Ronaldo//schießt 11m, Maus/Ente, )
- echtes 3D
- andere Eigenschaften (MiniGames)
- unterschiedliche Maps (verschiedene hintergründe, Röhre wie bei Flipper um in anderen Mapteil zu gelangen)
- Bossgegner (beweglich, HP-Anzeige, Minispiel-Kampf)
- Spawnrate der Items ändern
- Punkte nur bei einsammeln der Sprites (nicht bei Zerstörung)
- Soundtrack (bei Zerstörung, Items, Punkten etc)
